# Consultation Recording Manager

## Overview

A full-stack web application for managing consultation recordings.

Users can:

* Register and Login
* Upload consultation recordings
* View all uploaded recordings
* Delete recordings
* Manage recording metadata

## Tech Stack

### Frontend

* React
* React Router
* Axios

### Backend

* Node.js
* Express.js
* JWT Authentication
* Multer

### Database

* MongoDB

## Setup Instructions

### Backend

```bash
cd backend

npm install

npm run dev
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

### MongoDB

Ensure MongoDB is running locally on:

```bash
mongodb://127.0.0.1:27017/consultation_db
```
